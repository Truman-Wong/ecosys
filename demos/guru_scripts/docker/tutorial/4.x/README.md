# Content
This GSQL tutorial contains 

- Setup schema (model)
- Load data
- Query Examples


# Setup Schema
Copy [ddl.gsql](./script/ddl.gsql) to your container. 
Next, run the following in your container's bash command line. 
```
gsql ddl.gsql
```

# Load Data
Copy [load.gsql](./script/load.gsql) to your container. 
Next, run the following in your container's bash command line. 
```
gsql load.gsql
```
